# Home

The landing page for Scrunch Tools. Links to all available tools.

## What it does

A simple static HTML page that lists every tool with a description and a link. No build step required.

## Tools linked

| Tool | Path |
|---|---|
| Bulk Setup | `/bulk-setup/` |
| API Exporter | `/export/` |
| Slides Generator | `/slides/` |
| Response Analyzer | `/analyze/` |
| Looker Studio Template | External link |

## Development

Open `index.html` directly in a browser, or serve it with any static file server:

```bash
npx serve .
```
